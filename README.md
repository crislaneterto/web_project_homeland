# Tripleten web_project_homeland

TripleTen Art Gallery - De Pátria para Pátria

Este projeto é uma landing page responsiva para a Galeria de Arte TripleTen. O objetivo da página é celebrar a diversidade cultural dos profissionais de tecnologia, apresentando histórias e paisagens de diferentes partes do mundo, como Kentucky, Burundi, País de Gales e Ucrânia.
Tecnologias Utilizadas

    HTML5: Estruturação semântica do conteúdo.

    CSS3: Estilização e layout (via arquivo externo pages/index.css).

    Metodologia BEM: Utilizada para organizar as classes CSS e manter o código escalável.

    CSS Grid & Flexbox: Implementação de layouts complexos e responsivos. Utilização de grid-template-areas para organizar a hierarquia visual entre imagens, títulos e descrições.

Estrutura do Projeto

O projeto está organizado da seguinte forma:

    index.html: Arquivo principal com a estrutura da página.

    images/: Pasta contendo o logotipo e a imagem principal da seção lead.

    pages/index.css: Arquivo de estilos (esperado conforme a tag <link>).

    favicon.svg: Ícone da aba do navegador.

Seções da Página

    Header: Contém o logotipo da Galeria TripleTen.

    Lead: Apresenta o título principal, um subtítulo sobre a jornada épica e uma imagem de destaque.

    Intro: Introdução à galeria com uma citação de Joseph Beuys e um texto convidativo sobre a diversidade da comunidade.

    Conteúdo Enriquecido: Adição de destinos diversificados, incluindo Criccieth (País de Gales), Berea (EUA), Muramvya (Burundi) e o Galo da Madrugada em Recife (Brasil).

    Semântica Aprimorada: Cada destino foi estruturado dentro de um bloco .place, contendo títulos hierárquicos (h2), metainformações de autoria/artistas, imagens com textos alternativos (ALT) otimizados para acessibilidade e parágrafos descritivos.

    Footer: Rodapé com informações de direitos autorais.
